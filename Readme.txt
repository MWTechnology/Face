Установка
требуется версия python 3.6 (испытывалось на 3.6.7)
1) Скачай или клонируй всё содержимое 
2) Зайди через консоль в скаченную дирректорию 
3) введи в консоли команду pip install -r requirements.txt
4) при вводе в консоле данной команды python face_new_sosed.py вы запустите одновремменно и обучалку и сканер изображений



в дирректории knn_examples/train находятся файлы с помощью которых обучается программа (их можно добавлять) название папки это имя человека
в дирректории knn_examples/test находятся фотки на которых надо распознать людей
